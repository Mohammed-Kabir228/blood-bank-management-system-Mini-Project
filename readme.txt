# 🩸 Blood Bank Management System

A web-based Blood Bank Management System developed using PHP and MySQL to simplify the management of blood donors, blood donations, blood requests, and blood handovers. The system provides separate access levels for administrators and staff, ensuring efficient and secure blood bank operations.

## 🚀 Features

* Donor Management
* Blood Donation Records
* Blood Request Management
* Blood Handover Tracking
* User Management
* Admin & Staff Roles
* Responsive Dashboard
* AJAX-Based Operations
* Bootstrap UI Design

## 🛠️ Technologies Used

* PHP
* MySQL
* HTML5
* CSS3
* JavaScript
* Bootstrap
* jQuery
* AJAX

## 💻 Software Requirements

* XAMPP / WAMP / LAMP / MAMP
* Web Browser (Chrome, Firefox, Edge, Opera)
* Text Editor (VS Code, Notepad++, Sublime Text)

## 📦 Installation Guide

### Step 1: Download and Install XAMPP

Download and install XAMPP from the official website.

### Step 2: Extract Project Files

Extract the downloaded ZIP file using WinRAR or any archive tool.

### Step 3: Copy Project Folder

Copy the **Blood Bank System** folder and paste it into:

```text
xampp/htdocs/
```

### Step 4: Create Database

Open:

```text
http://localhost/phpmyadmin
```

Create a new database named:

```text
bloodbank_db
```

### Step 5: Import Database

Import the file:

```text
bloodbank_db.sql
```

from the SQL folder included in the project.

### Step 6: Run the Project

Open your browser and visit:

```text
http://localhost/bloodbank
```
